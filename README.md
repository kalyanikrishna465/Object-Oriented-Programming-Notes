### 🧠 Deep-Dive Fundamentals
Go beyond the basics. Master the "Why" behind the four pillars:
* **Abstraction** (Hiding Complexity)
* **Encapsulation** (Data Hiding)
* **Inheritance** (Code Reusability)
* **Polymorphism** (Dynamic vs Static)

### 🏗️ Java Internal Architecture
Visual and logical breakdowns of concepts that recruiters love to drill:
* **JVM vs JRE vs JDK:** The architectural "Formula" approach.
* **Stack vs Heap Memory:** A deep dive into where your data actually lives.
* **The Compilation Cycle:** From `.java` source code to Bytecode to Machine Code.

* **Exception Handling:** Hierarchy, Custom Exceptions, and the "Triple F" (Final/Finally/Finalize).
* **Multithreading:** Thread Life Cycles, Synchronization, and Inter-thread communication.
* **Generics:** Type Erasure, Bounded Types, and the PECS principle.
